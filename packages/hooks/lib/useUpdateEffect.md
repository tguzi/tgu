# useUpdateEffect

useEffect去初始化

#### Usage
<code>useUpdateEffect(effect, deps)</code>

#### Params
| Property | Description | Type | Default | Required |
| ---- | ---- | ---- | ---- | ---- |
| effect | 原本的effect回调 | Function | null | true |
| deps | 监听字段 | any | null | true |
